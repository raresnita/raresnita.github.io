# rarestein.github.io

## Description
This is the GitHub repository for my portfolio. It's a single-page website, built using [TailwindCSS](https://tailwindcss.com/) and [Font Awesome](https://fontawesome.com/).

## Features
- Responsive design
- Light / dark mode support
- Localisation in both English and Romanian
